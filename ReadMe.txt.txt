Todas las aplicaciones desarrolladas en esta demos se han realizado bajo
ASP.NET Core 1.0 y el editor de código Visual Studio Code que puede ser
descargado en https://code.visualstudio.com/

Para continuar se debe instalar VS Code y leer el fichero de "Instalación ASP.NET Core.txt"

Todos los ejemplos están explicados en la sesión grabada en Channel 9,
https://channel9.msdn.com/Events/NET-Conference/2016/Crea-tus-aplicaciones-web-con-SPA-Angular2-Web-Api-y-Visual-Studio-Code
